Conceito

É um framework JavaScript de código aberto mantido pela Google para a construção de SPA (sigla para Single Page Applications ou Aplicações de Página Única).

Forma de Utilização

Basicamente uma aplicação web construída em uma só página, na qual a interação e a navegação entre as sessões de uma página ocorrem de maneira a qual não é necessário recarregar a página em cada uma dessas mudanças.

Vantagens

  • O Framework, é extremamente robusto, ou seja, quase tudo que é preciso para uma aplicação se encontra nele nativamente.
  • Angular CLI, uma ferramenta que agiliza a criação das estruturas no projeto.
  • Html separado do JavaScript

Desvantagens

  • Muitos break change, o angular criou uma fama negativa por ter muitas mudanças nas suas atualizações que mudavam algumas sintaxes e estruturas que muitos tinham se acostumado a usar.
  • Mas nas versões mais recentes isso tem diminuído e ficado mais estável.
  • Dificuldade de migração entre versões. Isso mais entre as primeiras versões 1 pra 2.
